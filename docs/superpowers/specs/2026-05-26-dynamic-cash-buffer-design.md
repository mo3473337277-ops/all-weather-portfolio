# 动态现金补仓机制 — 设计说明

## 背景

网格搜索发现：风险平价（risk_parity）桶方法将 V3-B 回撤从 -6.29% 压到 -3.52%，但 CAGR 从 7.27% 降到 5.67%。核心矛盾是「太保守」——高波资产（商品/权益）在牛市涨最好，但风险平价给它们的权重最小。

纯静态现金档（85%/70% RP）只是等比缩小仓位，不改变这个矛盾。

本设计探索：**70% 风险平价核心 + 30% 动态补仓储备**，在保持低回撤底线的同时，用机动资金捕捉资产恐慌下跌后的反弹。

## 核心思路

```
┌─────────────────────────────────────────────┐
│  总资产 100%                                  │
│  ┌──────────────────────┐  ┌───────────────┐ │
│  │  70% 核心仓位          │  │  30% 现金储备   │ │
│  │  V3-B risk_parity     │  │  货币基金 ETF   │ │
│  │  月度再平衡            │  │  等待触发       │ │
│  └──────────────────────┘  └───────┬───────┘ │
│                                     │         │
│                          资产回撤 ≥ 阈值？     │
│                                     │         │
│                            ┌───────┴───────┐ │
│                            │  否：继续等待   │ │
│                            │  是：补仓该资产  │ │
│                            └───────────────┘ │
└─────────────────────────────────────────────┘
```

和静态现金档的本质区别：现金不是「永远不动吃利息」，而是「平时闲置，恐慌时出手」。

## 机制设计

### 触发条件（任一满足即触发）

监控 9 个资产各自从历史峰值的回撤幅度。当任一资产回撤 ≥ 阈值时，用现金储备买入该资产。

| 参数 | 候选值 | 说明 |
|------|--------|------|
| 补仓触发阈值 | -10%, -15%, -20% | 资产从自身峰值回撤超过此值触发 |
| 单次补仓比例 | 总资产的 3%, 5%, 7% | 每次触发时，从现金储备中划拨多少 |
| 最大补仓次数 | 3, 5, 不限 | 防止现金过早耗尽 |
| 同资产冷却期 | 30, 60, 90 天 | 同一资产两次补仓之间最少间隔 |

### 退出条件（何时卖回现金）

补仓买入的仓位独立追踪。当该资产从补仓价上涨 ≥ 退出阈值时，卖出补仓部分，资金回流现金储备。

| 参数 | 候选值 | 说明 |
|------|--------|------|
| 获利退出阈值 | +10%, +15%, +20% | 补仓部分盈利超过此值则卖出 |

### 核心仓位不受影响

- 70% 核心部分继续 V3-B risk_parity 月度再平衡，完全独立运行
- 补仓买入的仓位是「额外」的，不参与核心再平衡
- 核心和补仓仓位在净值计算时合并

## 回测逻辑概要

```
每个交易日：
  1. 核心 70%：正常 V3-B risk_parity 月度再平衡
  2. 检查 9 资产各自从峰值的回撤
  3. 若触发补仓条件 → 从现金划拨买入
  4. 检查已有补仓仓位的盈利状态
  5. 若触发退出条件 → 卖出补仓部分，资金回流现金
  6. 合并核心 + 补仓 + 剩余现金 = 总净值
```

## 参数网格（首轮探索）

| 参数 | 测试值 |
|------|--------|
| 补仓触发阈值 | -10%, -15%, -20% |
| 单次补仓比例 | 3%, 5% |
| 获利退出阈值 | +10%, +15% |
| 最大补仓次数 | 不限（靠冷却期控制频率） |
| 冷却期 | 60 天（固定） |

共 3 × 2 × 2 = 12 种组合，每种跑 100% 总仓位（核心 70% + 现金 30%）。

## 核心仓位参数（固定）

基于网格搜索最优结果：`bucket_method="risk_parity"`, `rp_window=90`, `max_w=0.30`, `min_w=0.02`

## 对比基线

- V3-B risk_parity 100% RP（无现金储备）：CAGR 5.74%, MDD -3.55%, Sharpe 1.56
- V3c 多元 100% RP（当前最优固定权重）：CAGR 7.45%, MDD -6.71%, Sharpe 1.26
- V3-B equal 100% RP（当前月度RP）：CAGR 7.27%, MDD -6.29%, Sharpe 1.14

## 成功标准

- MDD < -4.5%（优于纯 risk_parity 的 -3.55% 虽难，但至少优于 equal 的 -6.29%）
- CAGR > 6.5%（比纯 risk_parity 的 5.74% 有明显提升）
- 补仓机制在至少 3 个历史事件中被触发且有正贡献（如 2015 股灾、2018 大熊市、2022 股债双杀）

## 不在范围内

- 补仓加杠杆（不用融资）
- 做空或对冲工具
- 基于宏观指标的择时触发（只用价格回撤）

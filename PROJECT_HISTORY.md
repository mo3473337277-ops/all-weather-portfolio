## 3. 核心原则（Sperandeo 三层结构）

按优先级排序，不可颠倒。理论基础来自斯波朗迪《专业投机原理》的"保障资本 → 一致性获利 → 卓越回报"框架。

### 第一层：保障资本

任何改动前先考量风险：

| 实践 | 含义 |
|------|------|
| D_excess 尾部风险诊断 | 每次验证必须跑。D 显著为负（分位 > 97.5%）直接否决 |
| 墨菲定律 | 每个新信号必须有 fallback — 数据断了回退到基准权重 |
| 止损纪律 | 趋势过滤的本质是及时认赔，不在下跌趋势中死扛 |
| 不可摊平亏损 | 补仓 = 在亏损头寸上加码，方案 C 失败的根本原因 |

### 第二层：一致性的获利能力

| 实践 | 含义 |
|------|------|
| 奥卡姆剃刀 | 从最简方案开始，数据证伪才加复杂度 |
| 80/20 检验 | CAGR +0.3pp 或 MDD -1pp 以上才值得 |
| 休谟因果 | 必须有经济学逻辑支撑，消融实验验证增量 |
| 最小描述长度 | 参数响应面是平的 → 砍掉该参数 |

### 第三层：卓越的报酬

| 实践 | 含义 |
|------|------|
| HS300 AND 抄底 | 只在极端便宜且趋势企稳时适度加仓（1.8x），1:3 风报比 |
| 方案 C 的教训 | 储备金最有效的用途是留在核心里不动 |

### 方法论注脚：认知四步循环

斯波朗迪的认知四步循环（目标 → 行动 → 认知 → 改变）与优化闭环同构：

| 斯波朗迪 | 本项目 |
|----------|--------|
| 目标 — 知道希望追求的最后结果 | 问题定义 |
| 行动 — 拟定计划果断执行 | 假设 → 设计 |
| 认知 — 观察结果评估成败 | 验证 → 稳健性检验 |
| 改变 — 重新检查目标与行动 | 决策 → 实施 |

### 设计原则（八条，按流程对应）

| # | 原则 | 对应步骤 | 核心含义 |
|---|------|----------|----------|
| 1 | 基德林定律 | 问题定义 | 一句白话说清问题再动手 |
| 2 | 帕累托原则 | 问题定义 | 这是当前最大单一杠杆吗？不是 → 排后 |
| 3 | 没有免费午餐 | 假设 | 明确优化目标和牺牲项 |
| 4 | 奥卡姆剃刀 | 设计 | 能删什么而不是加什么？ |
| 5 | 休谟因果怀疑论 | 验证 | 消融实验验证增量，训练/验证切分防过拟合 |
| 6 | 最小描述长度 | 稳健性检验 | 参数响应面是平的 → 砍掉 |
| 7 | 墨菲定律 | 稳健性检验 | fallback 路径是什么？ |
| 8 | 透明度原则 | 实施 | 每步决策能解释给非技术人员 |

## 4. 优化历程

> 每次优化按闭环记录：问题 → 假设 → 设计 → 验证 → 稳健性 → 决策 → 结果。

### 4.1 回测期扩展（2008→2005）

**问题**：回测仅覆盖 2008-2026，缺少 CSI 300 最早两年数据（2005-2007 大牛市）。
**假设**：通过多数据源拼接可以前推到 2005-04-08（CSI 300 上线日）。
**设计**：三阶段债券缝合 + 黄金宏消费 proxy + 移除 div_idx 解决 panel dropna 约束。
**结果**：成功扩展到 2005，CAGR 因覆盖更多危机（2008 GFC）显著下降。

### 4.2 30Y 国债合成

**问题**：ETF 511130 2024-03 才上市，无法回测长期限债的历史表现。
**假设**：30Y 和 10Y 国债收益存在稳定久期关系，可用 10Y × 久期倍数近似。
**设计**：三阶段 — 久期乘数法(2005-2020) → 利差法(2020-2024) → 真实 ETF(2024+)。
**结果**：可回测 2005 年起的 30Y 国债配置。

### 4.3 策略精简：6→3 策略 + 砍无效择时层

**问题**：V3-A/V3-B 各有 3 层风控，其中 2 层从未触发，1 层在帮倒忙。
**假设**：全天候是被动战略配置，时序择时在多资产组合上无效或有害。
**设计**：砍掉趋势过滤的非触发层、回撤止损、波动率降仓、相关性断路器。
**验证**：A/B 对比 — 精简后 CAGR 持平或略升，MDD 略深但可接受。
**决策**：通过。V3-A 降格为纯固定权重，V3-B 保留纯分层 RP。
**结果**：6 策略减至 3 策略。教训：多资产分散组合上叠加时序择时，要么无效要么有害。

### 4.4 资产精简：移除 div_idx 和 soymeal

**问题**：div_idx 与 hs300 相关性 0.90（无独立分散价值），soymeal 自身 Sharpe 为负。
**假设**：移除冗余和负收益资产不会损害组合表现。
**设计**：直接移除，不调整其他参数。
**验证**：V3c / V3-B 三策略对比移除前后指标。
**结果**：无性能损失，资产更精简。V3c 从 9→6，V3-B RP 从 9→6，V3-B Con 从 9→7。

### 4.5 nonferr 趋势过滤

**问题**：nonferr 是有色金属，周期性强，趋势性下跌时持续拖累组合。回撤归因分析确认 nonferr 是头号元凶（11 次 top-3 回撤贡献）。
**假设**：nonferr < SMA 时清仓转入 credit，可在保留上涨弹性的同时避免主跌段。
**设计**：月度调仓时检查 nonferr 价格 vs 75d SMA，跌破则清仓转 credit。
**验证**：对比 4 种风控（趋势过滤/DD 刹车/波动率目标/相关性断路器），趋势过滤唯一有效。
**稳健性**：窗口扫描 30-150d，75d 为 sweet spot。
**决策**：通过。V3-B 和 V3-B Con 纳入，V3c 后续改为月度调仓后同样纳入。
**结果**：V3-B MDD -4.89%（-1.40%），CAGR +0.45%。最关键的单一项风控改进。

### 4.6 Gold 趋势过滤

**问题**：2013 年 taper tantrum + 中国钱荒双杀，gold/nonferr/hs300 同时暴跌，V3-B RP MDD -10.34%。
**假设**：gold < 75d SMA 时清仓转 credit，可避免黄金主跌段。
**设计**：月度检查 gold vs 75d SMA，跌破清仓。
**验证**：2013 年从 -3.52%→+3.72%，MDD -10.34%→-7.35%。
**决策**：通过。仅用于 V3-B RP。
**结果**：MDD 跌破 -10% 心理关口被修复。

### 4.7 SP500 Trend Filter

**问题**：美股暴跌期仍满仓 SP500，2008 年仅 +0.13%。
**假设**：SP500 < 120d SMA 时清仓转 credit，可避免美股主跌段。
**设计**：月度检查 SP500 vs 120d SMA，跌破清仓。
**验证**：2008 年 +0.13%→+11.53%。MDD -12.37%→-9.48%。
**稳健性**：80/20 验证 ΔCAGR -0.22pp，可接受。
**决策**：通过。仅用于 V3-B RP。
**结果**：第三个趋势过滤器，三者互补（nonferr 周期品/gold 避险品/sp500 权益）。

### 4.8 HS300 抄底信号优化（四阶段）

**问题**：HS300 单纯价格回撤抄底胜率不够，2014-09 卖飞（牛市前夜退出）。
**假设**：加入 PE 估值作为二次确认（AND 逻辑），可过滤假信号。
**设计**：四阶段依赖优化 — 信号频率→独立性→退出逻辑→参数网格搜索(108 组合)。
**验证**：80/20 切分，训练集调参，验证集锁定。
**关键发现**：
- PE（col7 TTM 中位数）替换 col2（等权静态PE）是最大修复
- PE 退出 50→70%ile + DD 退出 peak-12%→peak-15%
- PE 和 DD 是层级关系（PE 是 DD 的子集），非平行独立
**结果**：V3c CAGR +0.97%，V3-B RP CAGR +1.26%（最大受益者），V3-B Con CAGR +0.81%。

### 4.9 D_excess 尾部风险诊断

**问题**：如何量化验证三策略没有隐藏尾部风险？
**假设**：实际几何超额与正态假设理论值的偏差 D 能捕捉偏态/肥尾。
**设计**：G_real = CAGR - rf，G_theo = SR_true × σ - σ²/2，D = G_real - G_theo。参数化 Bootstrap 10000 次检验显著性。
**关键陷阱**：不能用几何 Sharpe 代入（D 退化为 σ²/2 恒等式）。必须从日度超额收益算术均值算 SR_true。
**结果**：三策略 D 均≈0，分位≈50%，零隐藏风险证据。D 指标通过诊断。

## 5. 失败记录

> 和成功同样重要的知识资产。每条记录防止重复尝试。

| 日期 | 方向 | 问题 | 失败原因 |
|------|------|------|----------|
| 2026-05-26 | 方案 C 动态补仓 | 想用 30% 储备逢低补仓提收益 | 储备金机会成本 > 择时收益。补仓放大回撤（接飞刀），纯 100% RP 碾压所有参数组合 |
| 2026-05-26 | V3e/V3f 集中度变体 | 想通过更集中提高 CAGR | 集中反而伤害组合，Sharpe 2.60→2.37 |
| 2026-05-26 | 波动率降仓 + 相关性断路器 | V3-A 三层风控 | 相关性断路器 11 年 0 触发，波动率降仓在反弹年降仓反害回报 |
| 2026-05-26 | HS300 trend filter | 想加第四个趋势过滤器 | CAGR -0.31pp，得不偿失 |
| 2026-05-26 | Bucket risk parity | 想桶间按风险分配而非等权 | CAGR -0.13pp，无改善 |
| 2026-05-27 | HS300 dip-buying（旧版） | 价格回撤抄底 | 训练集显著但验证集 MDD 翻倍 — 过拟合典型案例 |
| 2026-05-28 | 关 nonferr 趋势过滤 | 想简化 | SMA120 和 gold dip 不可删，否决 |
| 2026-05-28 | Gold dip-buying（V3c/Con） | 黄金回撤抄底 | 2013 年接飞刀放大亏损，关闭后零负收益年份 |

**核心教训**：
- 每增加一层复杂度，边际收益递减，维护成本递增
- 多资产分散组合上叠加时序择时，要么无效要么有害
- 训练/验证切分是防范过拟合的最低防线
- 任何优化先问"能删什么？"而非"能加什么？"

## 6. 关键结论

### 权重方法论
- **分层 RP 的桶结构决定收益上限。** 5 桶优于 4 桶（10Y/30Y 分拆后 CAGR +0.41%）。
- **bucket_method 是主导参数。** risk_parity vs equal 决定 MDD/CAGR tradeoff。不存在两者兼得的参数。
- **固定权重意外难被超越。** 动态调整的噪音往往大于信号。

### 风控机制
- **nonferr 趋势过滤是最有效的单一风控。** 硬过滤（zero out）始终优于软过滤（halving）。nonferr → credit 是最优重定向。
- **趋势过滤需要月度或更高频调仓。** V3c 半年调仓加趋势过滤无效（来不及反应）。
- **风控叠加会过度过滤。** 单一机制 + 适度参数是最优解。
- **多资产趋势过滤无额外增益。** 仅 nonferr 值得过滤 — 周期性强、趋势性明显。

### 参数规律
- **20d 窗口优于 60d。** 更敏捷，15d 以下会不稳定。
- **max_w=0.25 是逆波动率最优上限。** 0.20 太集中，0.30 太分散。
- **min_w 几乎无影响。** 逆波动率自然产生的权重很少触及下限。

### 抄底信号
- **PE 和 DD 是层级关系，非平行独立。** PE 是 DD 的子集（P(DD\|PE)=89.5%），作为二级确认有价值。
- **PE 退出阈值是最大单一改进杠杆。** 50→70%ile 防止卖飞。
- **DD 信号不能单独使用**（会过拟合），但作为 PE 的二级确认贡献 +0.72% 验证 CAGR。
- **PE 数据列的正确选择至关重要。** col7（TTM 中位数）vs col2（等权静态PE）在 2015 股灾时表现天差地别。

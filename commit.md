# Commit 说明

## 本次改动概述

### 1. 数据拉取层（`allweather/fetch.py`）

- **腾讯证券备用源**：新增 `_fetch_idx_tx()`，对 `idx` 和 `idx_em` 类型，当主接口（新浪/东财）数据截止过早或断连时，自动降级到腾讯证券接口
- **数据覆盖范围检查**：`fetch_one` 拿到数据后判断最新日期是否覆盖请求终止日，不足 30 天则主动降级，避免静默返回空数据
- **特殊符号替换**：将 `📥 📅 ▸` 等 emoji 替换为 ASCII，兼容 Windows GBK 控制台

### 2. 数据加载层（`allweather/data.py`）

- **索引排序修复**：`load_panel()` 在 `.loc` 切片前加 `sort_index()`，解决多源数据拼接后索引不单调导致的 `KeyError`

### 3. 流程编排（`allweather/pipeline.py`）

- **特殊符号替换**：将所有 `✓ ✅` 替换为 `ok`，兼容 Windows GBK 控制台

### 4. 报告层（`allweather/reports.py`）

- **特殊符号替换**：将 `▼ 💡` 替换为 ASCII，兼容 Windows GBK 控制台

### 5. 入口（`main.py`）

- **特殊符号替换**：将 `❌` 替换为 `[ERROR]`

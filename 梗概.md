# 代码梗概 — 快速定位指南

> 修改代码前先读这个，再去看具体文件。免去每次从头啃 14 个 py 文件。

## 一句话架构

`main.py` → `pipeline.py`（6步流水线）→ 各模块各司其职。所有配置在 `config.py`。

## 文件职责 & 关键函数

| 文件 | 职责 | 关键函数/变量 |
|------|------|--------------|
| `allweather/config.py` | 常量+资产元信息 | `ASSETS`, `BUCKET_GROUPS`, `CASH_TIERS`, `BACKTEST_START/END`, `REBAL_FREQ`, `REBAL_THRESHOLD` |
| `allweather/data.py` | 从 data/ 读 CSV，缝合 ETF 上市前替代数据，输出 9 资产面板 | `load_panel()` → DataFrame, `synthesize_bond_30y()`, `stitch_series()` |
| `allweather/fetch.py` | 用 akshare 拉数据（可选） | `fetch_all(force=False)` |
| `allweather/portfolios.py` | **V3c 权重定义 + 策略标签** | `WEIGHTS` (dict: V3c→{asset:w}), `PORTFOLIO_TAGS`, `get_weights()` |
| `allweather/backtest.py` | V3c 回测引擎：半年+3%双触发再平衡 | `backtest(weights, rets, cash_ratio)` → (nv, n_rebal) |
| `allweather/strategy_b.py` | V3-B 分层风险平价(20d+趋势过滤) / 保守增强(逆波动率+趋势过滤) | `backtest_b(rets, ...)` → (nv, n_rebal) |
| `allweather/risk.py` | 风控原语库（pipeline 只用了 HRP 和逆波动率） | `hierarchical_rp_weights()`, `inverse_vol_weights()`, `trend_filter()`, `drawdown_stop()`, `vol_target_scale()`, `correlation_breaker()` |
| `allweather/stats.py` | 统计指标 + 蒙特卡洛 | `perf_metrics(nv)` → dict, `yearly_returns()`, `event_returns()`, `block_bootstrap()` |
| `allweather/reports.py` | 控制台打印 + JSON/CSV 保存 | `print_perf_table()`, `save_nv_curves()`, `save_summary_json()` |
| `allweather/excel_export.py` | 多 sheet Excel 报告（需要 openpyxl） | `save_excel_report()` |
| `allweather/markdown_report.py` | Markdown 报告 | `save_markdown_report()` |
| `allweather/pipeline.py` | 6 步流水线编排 | `run_full_pipeline()`, `step_1~6` 可单独调用 |
| `main.py` | CLI 入口 | argparse → `run_full_pipeline()` |

| 想改什么 | 去哪里 |

| 现金档位 | `allweather/config.py` → `CASH_TIERS` |
| 资产/桶定义 | `allweather/config.py` → `ASSETS`, `BUCKET_GROUPS`, `ETF_META` |
| 添加新资产 | `config.py` (+ ASSETS/ETF_META/BUCKETS) + `data.py` (+ load逻辑) + `data/` (放CSV) + `fetch.py` (+ 抓取) |
| 30Y 国债合成逻辑 | `allweather/data.py` → `synthesize_bond_30y()` |
| 数据缝合（ETF上市前替代） | `allweather/data.py` → `stitch_series()` |
| 新增策略（固定权重类） | `portfolios.py` (+权重) + `pipeline.py` (step_2中跑回测) |
| 新增策略（动态权重类） | 新建 strategy_xxx.py + `pipeline.py` 引入 |
| 删除策略 | `portfolios.py` (+WEIGHTS/TAGS) + `pipeline.py` (step_2移除) + `markdown_report.py` (+notes) + `docs/index.html` (+表格) |
| 统计指标公式 | `allweather/stats.py` |
| Bootstrap 参数 | `allweather/config.py` → `BOOTSTRAP_*` |
| 控制台输出格式 | `allweather/reports.py` |
| Excel 报告 sheet | `allweather/excel_export.py` |
| Markdown 报告内容 | `allweather/markdown_report.py` |
| GitHub Pages 文档 | `docs/index.html` |
| CI sanity check 阈值 | `.github/workflows/backtest.yml` |

## 数据流（一次完整回测）

```
data/*.csv → data.load_panel() → (panel, rets)
  → backtest.backtest() [V3c] / strategy_b.backtest_b() [V3-B]
  → stats.perf_metrics() + yearly_returns() + ...
  → stats.block_bootstrap()
  → reports (控制台) + excel_export (Excel) + markdown_report (MD)
  → output/nv_curves.csv, summary.json, weights.csv, report.xlsx, report.md
```

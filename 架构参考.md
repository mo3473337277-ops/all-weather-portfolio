# 架构参考

模块结构、流水线、依赖关系。按需读取。

---

## 目录结构

```
main.py → pipeline.run_full_pipeline()
allweather/
  config.py         所有常量
  data.py           加载 + 合成 30Y 国债
  fetch.py          通过 akshare 拉数据
  backtest.py       统一回测引擎（backtest / backtest_iv）
  strategy_b.py     backtest_b 向后兼容包装
  risk.py           逆波动率 / 分层风险平价 / 趋势过滤（部分仅库函数）
  stats.py          指标 / Bootstrap / D_excess
  reports.py        控制台输出
  excel_export.py   Excel 报告
  markdown_report.py Markdown 报告
  pipeline.py       6 步编排
```

## 6 步流水线

1. 加载 9 资产面板 → 2. 跑 3 策略 × 3-4 现金档 = 11 回测 → 3. 算衍生指标 + D_excess 显著性 → 4. Block Bootstrap (1000×5yr, 21d block) → 5. 打印控制台 → 6. 保存 CSV/JSON/Excel/Markdown/图表

## 模块依赖

```
                     main.py
                        |
                  (延迟导入)
                     /  |  \
                    /   |   \
           fetch.py  pipeline.py  experiment_log.py
               |       /  |  \  \  \
               |      /   |   \  \  \
               v     v    v    v  v  v
   config.py  <--- data.py  backtest.py  stats.py  reports.py  update_docs.py  charts.py
      ^         (lazy)         |            ^
      |                        |            |
      |                    risk.py          |
      |                    ^  ^             |
      |                    |  |             |
      |           strategy_b.py  |          |
      |                |         |          |
      +--- excel_export.py       |          |
      +--- markdown_report.py    |          |
      +--- rebalance.py ---------+          |
      +--- grid_search_b.py ------+---------+
      +--- explore_improvements.py ---------+
```

**分层：**
- **第 0 层**（无 allweather 依赖）：config.py, risk.py, experiment_log.py
- **第 1 层**（仅依赖 config）：data.py, fetch.py, stats.py, charts.py, reports.py, excel_export.py, markdown_report.py, update_docs.py
- **第 2 层**（依赖 config + risk/data）：backtest.py, rebalance.py
- **第 3 层**（依赖 backtest）：strategy_b.py
- **第 4 层**（编排器）：pipeline.py
- **入口**：main.py（延迟导入 fetch, pipeline, experiment_log）

> 详细分析见 `项目.md`。
